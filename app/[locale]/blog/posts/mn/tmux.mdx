Програм хөгжүүлэгчдийн хувьд терминал ашиглах нь хамгийн бүтээмжтэй, юуг ч хийж болхоор хөрвөмтгий байдаг билээ. Мөн терминал-аа нээгээд суухад яг л хакер/шидтэн мэт л мэдрэмжийг цацруулж чаддаг нь нууц биш. Миний өдөр тутмын командын мөр ашиглахад хамгийн чухал хэрэгслүүдийн нэг бол **tmux** гэж хэлэхэд буруудахгүй биз ээ~!!

![tmux](/blog/tmux/tmux.png)

# tmux гэж юу вэ?

tmux бол терминал мультиплексер юм. Энэ нь нэг терминал дээр хэд хэдэн програм хооронд шилжих, тэдгээрийг салгах, өөр терминалд дахин холбох боломжийг олгодог. tmux-ийг ихэвчлэн гар болон хулганы хэрэглээг тасалдуулахгүйгээр урт хугацааны програмууд, тухайлбал компиляци эсвэл демон процессуудыг ажиллуулахад ашигладаг. tmux-ийн хамгийн түгээмэл хэрэглээ нь терминал дээр суурилсан текст засварлагчид бөгөөд энэ нь нэг терминал дээр файл засах, нөгөө терминал цонхонд өөр файл нээх боломжийг олгодог.

tmux нь үндсэн хоёр том боломжийг санал болгодог:

1. Терминал дээрх цонхны удирдлага
2. Session удирдлага

Хэрэв та GNU Screen-тэй танил бол энэ бүгд танил сонсогдож магадгүй. tmux-ийг Screen-ийн илүү хялбар, бас илүү хүчирхэг хувилбар гэж бодож болно.

Концепт-уудыг орхиод практик руу орцгооё!

# Суулгах

Ихэнх үйлдлийн системд tmux суулгах нь маш энгийн:
`sudo apt-get install tmux` (Ubuntu, WSL болон түүнтэй төстэй) эсвэл `brew install tmux` (MacOS) гэж бичихэд л хангалттай.

# Анхны Session Эхлүүлэх

Анхны Session хувьд tmux-ийг шинэ session эхлүүлнэ:

```
tmux
```

Энэ нь доод талд ногоон өнгөтэй статусын мөртэй шинэ tmux session үүсгэнэ:

![tmux_green_status_bar](/blog/tmux/tmux_green_status_bar.png)

# Панел Хуваах

Одоо та session ажиллуулж байгаа учир терминалаа олон панел болгон хуваах боломжтой болсон.
Панелууд нь tmux цонхон дотор хуваагдсан хэсгүүд юм - нэг цонх олон панелтай байж болно.
Та одоогийн панелээ хоёр хэсэг болгон `Ctrl-b %` (`Ctrl-b` нь урьдчилсан түлхүүр бөгөөд үүнийг `~/.tmux.conf` файл дотор өөрчилж болно) ашиглан хуваах боломжтой.
Энэ нь таны цонхыг зүүн ба баруун талд хоёр панел болгон хуваана:

![tmux_split_panes](/blog/tmux/tmux_split_horizontal.png)

Панел хооронд шилжихийн тулд `Ctrl-b <сум түлхүүр>` ашиглана.

**Панелийн хэмжээг өөрчлөх** Панелийн хэмжээг өөрчлөхдөө:

```
Ctrl-b + Alt + <сум түлхүүр>
```

# Session Салгах

Sesion-ийг салгаж, background горимд ажиллуулахын тулд `Ctrl-b d` ашиглана.
Одоо байгаа session-д **дахин холбогдохын** тулд эхлээд ажиллаж буй session-үүдийг жагсаана:

```
tmux ls
```

Дараа нь тохирсон session-д холбогдоно:

```
tmux attach -t <session-нэр>
```

# Цонх Үүсгэх

Хэрэв та шинэ цонх үүсгэхийг хүсвэл `Ctrl-b c` ашиглаж болно. Энэ нь доод талдаа бүх ногоон статусын самбартай шинэ цонх үүсгэнэ:

![tmux_green_status_bar](/blog/tmux/tmux_create_window.png)

# Цонх Хооронд Шилжих

Цонх хооронд шилжихийн тулд дараах командуудыг ашиглана:

```
Ctrl-b n # Дараагийн цонх руу шилжих
Ctrl-b p # Өмнөх цонх руу шилжих
Ctrl-b w # Цонхны сонгогч нээх
```

# Гүйлгэх

Хэрэв та tmux дээр дээш доош гүйлгэхэд ямар нэгэн хүндрэлтэй тулгарсан бол дараах товчлууруудыг дарж үзээрэй:

```
Ctrl-b [
```

- Энэ нь tmux-ийг **copy mode/хуулах горим**-д оруулж, хэрэглэгчид сум түлхүүрээр дээш/доош шилжих боломж олгодог.

# Цонхны Нэр Өөрчлөх

Хэрэв та цонхны нэрийг өөрчлөхийг хүсвэл `Ctrl-b ,` (энд `,` нь таслалын түлхүүр) ашиглана. Энэ нь шинэ цонхны нэр оруулах боломжтой цонх гаргаж ирнэ.

# Session Удирдлага

Хэрэв та session-ээ дуусгахыг хүсвэл **дотор нь ажиллаж буй бүх процессыг хаах** эсвэл дараа ашиглахын тулд background горимд ажиллуулж үлдээж болно.

Одоогийн session-ээ салгахын тулд `C-b d` ашиглана. Мөн `C-b D` ашиглан аль session-ээ салгахыг сонгох боломжтой.
Session-ээс салгах нь тухайн session дээр хийж байсан бүх зүйлийг background горимд шилжүүлж үлдээнэ гэсэн үг. Энэ нь та дараа энэ session-г буцааж ашиглаж болно гэсэн үг.

Session-д дахин холбогдож, өмнө орхисон газраасаа үргэлжлүүлэхийн тулд эхлээд аль Session-д холбогдохоо тодорхойлох хэрэгтэй. Одоо ажиллаж буй session-үүдийг

```
tmux ls
```

ашиглан жагсааж харна.

Энэ нь таны жишээнд бүх ажиллаж буй session-үүдийн жагсаалтыг харуулах бөгөөд ийм байж болно:

*0: 2 windows (created Sat Aug 15 17:55:34 2015) [199x44] (detached)*

Аль session-д холбогдохоо зааж өгнө:

```
tmux attach -t 0
```

Энд `-t 0` нь tmux-д аль session-д холбогдохыг хэлж өгч буй параметр юм. "0" нь таны `tmux ls` гаралтын эхний хэсэг юм.

Хэрэв та session-дээ илүү утга учиртай нэр өгөхийг хүсвэл (0-ээс эхэлсэн тоон нэрний оронд) дараагийн сэшнээ

```
tmux new -s database
```

ашиглан үүсгэж болно.

Мөн одоо байгаа session-ий нэрийг өөрчилж болно:

```
tmux rename-session -t 0 database
```

Дараа удаа тэр session-д холбогдохдоо та зүгээр л `tmux attach -t database` гэж бичнэ.
Хэрэв та олон session зэрэг ашиглаж байгаа бол энэ командууд таны ажлийг хялбарчилж өгөх болно.

# Session Устгах

Тодорхой session-ийг устгахын тулд дараах командыг ашиглана:

```
tmux kill-session -t <session-нэр>
```

Хэрэв та **бүх tmux session-ийг хүчээр зогсоохыг** хүсвэл дараах командыг ашиглана:

```
tmux kill-server
```

⚠️ **Анхааруулга:** Энэ нь салгасан session-үүдийг оролцуулан **бүх ажиллаж буй session-үүдийг унтраах** болно.
