Програм хангамжийн инженерийн хувьд би өдөр бүр Git болон GitHub ашигладаг ба өдөр тутамдаа терминал дээр Git CLI командуудыг ашигладаг байсан. Энэ нь Github Desktop гэх мэт Graphic Interface-үүдээс илүү хурдан, хялбар, хөнгөн ажилдаг санагддаг байлаа. Гэвч зарим нэг илүү төвөгтэй эсвэл эрсдэлтэй үйлдлүүд хийх шаардлагатай үед асуудлуудтай тулгарч эхэлж байв. Github Desktop гэх мэт tools-ийг ашиглах гэсэн боловч нэг л том нүсэр зай эзэлсэн удаан санагдаж байв.

Нэг л өдөр үзэх дуртай streamer-ээсээ Lazygit-ийг ашиглаж байхыг нь олж хараад туршиж үзмээр санагдав. Lazygit ашигласнаар би CLI командууд шиг хурдан ажиллах боломжтой болсон бөгөөд үүн дээр нэмээд UI-н ойлгомжтой байдал нь их хялбар санагдсан. Lazygit-ийн бас нэг онцлог зүйл бол бүрэн гүйцэд гарын товчлууруудаар удирдагддаг — хулгана огт ашиглах шаардлагагүй гэсэн үг.

# Lazygit ашиглах нь

Та терминалын UI доторх панелүүдийн хооронд суман товчлуурууд эсвэл `H`, `J`, `K`, `L` гэх мэт vim хэв маягийн товчлуурууд ашиглан амархан шилжиж болно. Зарим панел хэд хэдэн табтай байдаг бөгөөд эдгээрийг `[` болон `]` товчлууруудаар сольж болно. Хамгийн хэрэгтэй товчлуур бол `?` — энэ нь тухайн панел эсвэл таб дээр ямар үйлдэл боломжтойг харуулсан тусламжийн цэсийг гаргаж өгдөг.

![lazygit](/blog/lazygit/lazygit.png)

Lazygit-ийн товчлуурууд ойлгоход маш хялбар — ямар нэгэн үйлдэл хийх гэж байхад яг л тэр үйлдлийг илэрхийлсэн товчлуурыг дархад л хангалттай. Жишээ нь, `c` дарвал файлын панел дээр өөрчлөлтүүдийг commit хийх, `n` шинэ branch эсвэл worktree үүсгэх (контекстээс хамаарна), `p` дарвал push хийх, `o` дарвал commit эсвэл файл нээх гэх мэт. Хэрвээ яг юу хийхээ мэдэхгүй байвал зүгээр л `?` дархад тухайн контекстэд тохирсон тусламжийн цэс гарч ирнэ.
